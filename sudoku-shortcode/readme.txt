Fast and easy integration of sudoku puzzles into your blogs, with some parameters. 

== Description ==

This plugin allows you to include sudoku puzzles in your blogs with a simple shortcode. The sudoku puzzles can be inserted everywhere: in your theme files, posts and pages.

Parameters:

*	Size of the sudoku puzzle: 9x9, 10x10, 12x12, 16x16.
* 	Difficulty level: easy, medium, hard, expert.
*   Width (and height) in pixels (not less than 450 px).
*  	Background color.
* 	Available in english and french.

For more details and examples visit the <a href="http://www.live-sudoku.com/site-perso/wordpress">live sudoku plugin page</a> or the <a href="http://www.le-sudoku.fr/site-perso/wordpress">sudoku plugin page</a> if you speak french.

== Installation ==

1. Upload 'Sudoku ShortCode' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Sudoku Shortcode Options page under settings and save your prefered options.
4. Use the Sudoku shortcode to insert sudoku puzzles into your posts/pages.

The <a href="http://www.live-sudoku.com">sudoku puzzles</a> are provided by <a href="http://www.le-sudoku.fr">Le sudoku</a>.

== How to use ==

Visit <a href="http://www.live-sudoku.com/site-perso/wordpress">sudoku in your Wordpress blog</a> or if you speak french <a href="http://www.le-sudoku.fr/site-perso/wordpress">sudoku sur votre blog Wordpress</a>.
